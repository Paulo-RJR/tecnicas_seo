One Page Site.

Ideal para negócios de paqueno e médio porte o "one site" é um site onde você inclue todas as informações que o visitante precisa de forma objetiva e criativa em uma única página.

Wireframe.

Wireframe é uma representação visual da estrutura básica de uma página. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essêncial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o wireframe desta página.
